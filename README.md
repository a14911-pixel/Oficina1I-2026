# 🐍 Exercícios de Python — Estruturas Condicionais

Este repositório contém um conjunto de exercícios em **Python**, focados em estruturas condicionais (`if`, `elif`, `else`). Estes exercícios fazem parte da minha aprendizagem no curso **Técnico de Gestão e Programação de Sistemas Informáticos**.

O objetivo destes programas é treinar a lógica de programação, a tomada de decisões e o uso de operadores relacionais e lógicos.

---

## 📘 Conteúdos abordados
- Verificação de números positivos/negativos  
- Maioridade  
- Par e ímpar  
- Comparação de valores  
- Validação de password  
- Classificação de notas  
- Classificação etária  
- Múltiplos de 3 e 5  
- Autenticação simples  
- Intervalos numéricos  
- Simulação de levantamento bancário  
- Determinação do maior entre vários números  
- Cálculo de IMC  
- Cálculo de descontos  
- Verificação de ano bissexto
